# CFW Finance Skill

This child skill package focuses on deploying and validating Huawei Cloud Firewall for financial institutions such as banks, insurers, fintech platforms, and payment systems. It is intended for security foundation scenarios where compliance-driven controls, IPS protection, ACL policy design, logging, and alarm coverage must be delivered in a repeatable way.

## Included Assets

- [SKILL.md](./SKILL.md): Main skill definition, workflow, core rules, and validation gates
- [references/](./references): Compliance mapping, ACL patterns, IPS tuning, logging guidance, and troubleshooting
- [scripts/](./scripts): Helper scripts for preflight, deployment, ACL setup, IPS, logging, alarms, and validation
- [agents/](./agents): Agent configuration used by the skill package

## Typical Use

- Configure Huawei Cloud Firewall for regulated financial workloads
- Apply finance-specific ACL and IPS policies
- Enable logging and alarm paths for audit readiness
- Validate compliance-oriented firewall controls before production rollout
