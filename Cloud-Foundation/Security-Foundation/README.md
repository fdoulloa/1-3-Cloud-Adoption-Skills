# Security Foundation

This use case defines the core security baseline required for cloud adoption, including protection controls, containment, anti-tampering, audit, and compliance. It creates a secure operating posture before broader application and data workloads are introduced.

## Typical Skill Areas

- Security baseline definition
- Intrusion prevention and attack containment
- Data leakage prevention
- Anti-tampering controls
- Audit and compliance alignment

## Expected Outputs

- Security baseline package
- Control mapping and hardening plan
- Audit and compliance checklist
- Risk findings and remediation actions
