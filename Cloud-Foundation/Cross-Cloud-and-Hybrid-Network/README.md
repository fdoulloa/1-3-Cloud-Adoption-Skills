# Cross-Cloud and Hybrid Network

This use case covers network connectivity between on-premises environments, Huawei Cloud, and other clouds. It supports hybrid and cross-cloud delivery scenarios where stable, secure, and governable connectivity is required.

## Typical Skill Areas

- Cross-cloud network architecture design
- VPN, dedicated connection, and interconnect planning
- On-premises to cloud connectivity
- Cloud to cloud connectivity
- Hybrid cloud collaboration patterns

## Expected Outputs

- Target network architecture
- Connectivity and routing design
- Security and segmentation baseline
- Connectivity validation plan
