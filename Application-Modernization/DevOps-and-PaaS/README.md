# DevOps and PaaS

This use case focuses on modern delivery pipelines and application platform integration, including CI/CD, artifact management, test automation, and platform service adoption. It helps teams improve delivery speed and consistency after migration or modernization.

## Typical Skill Areas

- CI/CD pipeline design
- Artifact management
- Test automation
- PaaS platform integration
- Release workflow standardization

## Expected Outputs

- DevOps workflow design
- CI/CD and test automation blueprint
- Platform integration plan
- Delivery metrics and validation checklist

## Included Skills

- [Karmada K8s Switch Skill](./karmada-k8s-switch-skill/README.md): Build a local Karmada lab, deploy the verified `member1` and `member2` failover PoC, switch traffic between Kubernetes clusters, and cleanly tear the environment down.
