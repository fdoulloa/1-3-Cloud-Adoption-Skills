# Database Migration

This use case covers migration and modernization paths for enterprise databases, including commercial relational engines, open source databases, and NoSQL platforms. It also supports target-state decisions across managed and modern database services.

## Typical Skill Areas

- SQL Server migration planning
- Oracle migration planning
- Open source database migration
- NoSQL database migration
- Database modernization path analysis

## Expected Outputs

- Database assessment and target recommendation
- Migration architecture and tooling plan
- Cutover and rollback plan
- Data validation and performance checklist

## Included Skills

- [GaussDB Adaptation Skill](./GaussDB-Adaptation-Skill/README.md): Reuse a database migration skill for adapting SQL Server or PostgreSQL workloads to Huawei GaussDB and openGauss-compatible environments.
