# Datacenter Migration into Cloud

This use case covers the migration of datacenter resources into Huawei Cloud, starting from inventory and assessment through migration wave planning and execution alignment. It is the foundation for large-scale infrastructure migration programs.

## Typical Skill Areas

- Resource inventory and dependency capture
- Migration wave design
- Migration path planning
- Environment readiness assessment
- Migration execution coordination

## Expected Outputs

- Current-state inventory package
- Migration wave plan
- Target migration roadmap
- Readiness and execution checklist
