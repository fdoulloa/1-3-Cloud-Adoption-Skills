# SAP on Cloud

This use case focuses on moving SAP landscapes to cloud with strong attention to performance, high availability, backup, and disaster recovery requirements. It supports both infrastructure migration and operational design for business-critical SAP systems.

## Typical Skill Areas

- SAP infrastructure migration planning
- High-availability architecture design
- Performance and capacity planning
- Backup and disaster recovery design
- Operational readiness planning

## Expected Outputs

- SAP target architecture
- Performance and HA design package
- Backup and DR plan
- Operational readiness checklist
