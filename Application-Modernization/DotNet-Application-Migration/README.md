# DotNet Application Migration

This use case covers the modernization of .NET Framework and Windows-based workloads toward current runtimes, cloud-ready deployment patterns, and cloud-native architectures where appropriate.

## Typical Skill Areas

- .NET Framework to modern runtime assessment
- Windows workload migration planning
- Application compatibility analysis
- Refactoring path definition
- Target deployment model selection

## Expected Outputs

- Compatibility and gap assessment
- Runtime modernization plan
- Migration architecture recommendation
- Validation and rollback checklist
