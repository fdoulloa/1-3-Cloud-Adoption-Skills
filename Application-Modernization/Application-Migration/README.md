# Application Migration

This use case focuses on moving applications to Huawei Cloud with the right balance between speed and transformation. It covers pre-migration assessment, dependency analysis, and the design of migration and modernization paths.

## Typical Skill Areas

- Application assessment before migration
- Application dependency mapping
- Migration path analysis
- Refactoring readiness evaluation
- Target cloud operating model design

## Expected Outputs

- Application assessment report
- Dependency map
- Migration approach recommendation
- Transformation backlog and plan
