# Data Engineering for AI

This use case focuses on the data and prompt pipelines required to support AI systems, especially retrieval-augmented generation and enterprise knowledge services. It bridges raw data preparation with model-facing application quality.

## Typical Skill Areas

- Prompt engineering
- RAG workflow design
- Knowledge base construction
- Vector retrieval design
- Data processing pipeline design

## Expected Outputs

- AI data pipeline architecture
- Prompt and retrieval baseline
- Knowledge base design
- Quality and retrieval validation report
