# OpenHands Huawei MaaS

Configure OpenHands to use Huawei Cloud MaaS / ModelArts MaaS through an OpenAI-compatible endpoint.

This skill is intended for local OpenHands adoption, including:

- Running OpenHands Web GUI with Docker.
- Installing and running OpenHands CLI with `uv`.
- Setting `LLM_API_KEY`, `LLM_BASE_URL`, and `LLM_MODEL`.
- Using Huawei Cloud MaaS models such as `glm-5.1`.
- Verifying MaaS connectivity without exposing API keys.
- Troubleshooting OpenHands, Docker sandbox, CLI, and MaaS endpoint issues.
