# Grill With Docs

This README describes only the `grill-with-docs/` skill directory under `AI/AI-Coding`.

## Purpose

`grill-with-docs` is a design and planning interview skill. It challenges assumptions, sharpens domain terminology, and updates project language artifacts such as `CONTEXT.md` and ADRs as decisions are resolved.

## Directory Structure

```text
grill-with-docs/
  README.md
  SKILL.md
  ADR-FORMAT.md
  CONTEXT-FORMAT.md
```

## Files

### `SKILL.md`

The main skill definition. It runs a structured grilling session grounded in the repo's domain model and existing decisions.

### `CONTEXT-FORMAT.md`

Reference format for building or extending a project glossary in `CONTEXT.md`.

### `ADR-FORMAT.md`

Reference format for recording durable architectural decisions as ADRs.
