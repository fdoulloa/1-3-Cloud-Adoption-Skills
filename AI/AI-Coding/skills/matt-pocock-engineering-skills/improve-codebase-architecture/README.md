# Improve Codebase Architecture

This README describes only the `improve-codebase-architecture/` skill directory under `AI/AI-Coding`.

## Purpose

`improve-codebase-architecture` is an architecture review skill focused on deep modules, cleaner seams, stronger locality, and better AI navigability across an existing codebase.

## Directory Structure

```text
improve-codebase-architecture/
  README.md
  SKILL.md
  DEEPENING.md
  INTERFACE-DESIGN.md
  LANGUAGE.md
```

## Files

### `SKILL.md`

The main skill definition. It explores the codebase, identifies architectural friction, and proposes deepening opportunities using a consistent vocabulary.

### `LANGUAGE.md`

Defines the architecture terms this skill expects to use consistently, such as module, seam, depth, and leverage.

### `DEEPENING.md`

Reference material for turning shallow modules into deeper, more useful ones.

### `INTERFACE-DESIGN.md`

Reference material for exploring alternative interface shapes when deepening a module.
