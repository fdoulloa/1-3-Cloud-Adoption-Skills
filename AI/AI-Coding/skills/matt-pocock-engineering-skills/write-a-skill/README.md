# Write A Skill

This README describes only the `write-a-skill/` skill directory under `AI/AI-Coding`.

## Purpose

`write-a-skill` is a skill-authoring workflow for creating new reusable agent skills with proper structure, scope, and supporting references.

## Directory Structure

```text
write-a-skill/
  README.md
  SKILL.md
```

## Files

### `SKILL.md`

The main skill definition. It guides the author through scoping, naming, file layout, and reference design for a new skill.
