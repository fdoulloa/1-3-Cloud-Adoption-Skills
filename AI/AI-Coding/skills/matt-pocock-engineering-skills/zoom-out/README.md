# Zoom Out

This README describes only the `zoom-out/` skill directory under `AI/AI-Coding`.

## Purpose

`zoom-out` asks the agent to explain a local piece of code in the context of the wider system so decisions can be made with architectural awareness instead of isolated file reading.

## Directory Structure

```text
zoom-out/
  README.md
  SKILL.md
```

## Files

### `SKILL.md`

The main skill definition. It guides the agent to reconnect implementation details to broader module, data-flow, and system intent.
