# Handoff

This README describes only the `handoff/` skill directory under `AI/AI-Coding`.

## Purpose

`handoff` compresses the current session into a durable handoff note so another agent or engineer can continue work without replaying the entire thread.

## Directory Structure

```text
handoff/
  README.md
  SKILL.md
```

## Files

### `SKILL.md`

The main skill definition. It captures current state, decisions, open risks, and next steps without duplicating existing durable artifacts.
