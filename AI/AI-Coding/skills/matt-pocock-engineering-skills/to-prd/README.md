# To PRD

This README describes only the `to-prd/` skill directory under `AI/AI-Coding`.

## Purpose

`to-prd` turns the current conversation and repo context into a product requirements document suitable for issue tracking and execution planning.

## Directory Structure

```text
to-prd/
  README.md
  SKILL.md
```

## Files

### `SKILL.md`

The main skill definition. It synthesizes current context into a PRD without forcing a separate interview round.
