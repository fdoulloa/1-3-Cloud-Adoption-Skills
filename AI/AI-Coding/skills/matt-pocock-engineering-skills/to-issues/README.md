# To Issues

This README describes only the `to-issues/` skill directory under `AI/AI-Coding`.

## Purpose

`to-issues` breaks a plan, spec, or PRD into small, independently executable issues using vertical slices instead of horizontal technical layers.

## Directory Structure

```text
to-issues/
  README.md
  SKILL.md
```

## Files

### `SKILL.md`

The main skill definition. It explores the repo context, then turns a larger initiative into issue-sized implementation slices.
