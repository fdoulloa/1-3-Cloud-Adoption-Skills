# ChatBI and Intelligent Analytics

This use case focuses on natural language analytics experiences that allow business users to ask questions, retrieve metrics, and generate analysis narratives. It bridges BI consumption with AI-assisted reporting and decision support.

## Typical Skill Areas

- Natural language to analytics workflows
- Executive BI dialogue design
- Metric question answering
- Automated analysis report generation
- Semantic layer and prompt alignment
- Conversational BI with text-to-SQL

## Included Skills

- [Simple ChatBI Skill](./Simple-ChatBI-Skill/): Foundational conversational BI with text-to-SQL, auto visualization, multi-provider fallback, and synthetic data generation.

## Expected Outputs

- ChatBI solution pattern
- Metric and semantic design baseline
- Automated reporting workflow
- User validation and accuracy report
