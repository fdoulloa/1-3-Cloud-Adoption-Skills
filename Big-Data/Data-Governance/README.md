# Data Governance

This use case covers the governance mechanisms required to keep enterprise data standardized, trusted, secure, and discoverable. It spans data standards, quality, metadata, permissions, and asset catalog capabilities.

## Typical Skill Areas

- Data standards definition
- Data quality management
- Metadata management
- Data security and access governance
- Data asset catalog design

## Expected Outputs

- Governance framework
- Quality and metadata baseline
- Access and security model
- Data catalog and operating checklist
