# Databricks to Huawei Cloud Skill

This Level 2 folder imports a reusable migration skill package for Databricks-to-Huawei Cloud work. It focuses on migrating Databricks tables, notebooks, SQL warehouse flows, and Spark pipelines toward Huawei Cloud patterns such as OBS, MRS Spark, and Hive.

## Included Assets

- [SKILL.md](./SKILL.md): Main skill definition, workflow, and default deliverables
- [references/](./references): Migration patterns, parity validation, and common pitfalls
- [scripts/](./scripts): Generic migration and validation templates
- [agents/](./agents): Agent configuration used by the skill package

## Typical Use

- Analyze Databricks source workflows
- Map source behavior to Huawei Cloud target patterns
- Generate sanitized migration templates
- Validate result parity and execution behavior
